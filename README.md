# 🚀 URL_SHORTENER

**Turn long URLs into short, shareable links in seconds!**  
Built with **Go** for the backend and **HTML/CSS** for a sleek, responsive frontend. Perfect for learning full-stack development or adding a practical project to your portfolio.  

---

## ✨ Features
- 🔗 Generate unique short URLs for any long URL   
- 🎨 Responsive and clean frontend design  
- 🗄️ Lightweight in-memory storage (can upgrade to a database later)  

---

## 🛠 Tech Stack
- **Backend:** Go (Golang)  
- **Frontend:** HTML & CSS  

---

## ⚡ How It Works
1. Open the web interface in your browser 🌐  
2. Enter a long URL 📝  
3. Click **"Shorten"** → get a unique short URL 🎯  
4. Share the link — redirects instantly to the original URL 🚀
